# 2026.5.27, 3일차 - 블록 떨어뜨리기
### 할 일: 
블록을 떨어뜨려서 블록이 쌓이도록 만들 것 이다.

### 한 일:
스페이스바를 눌렀을 때 블록의 위치가 아래 위치로 변하게 만들었다.

### 배운 점:
한 프로그램 내에서 무한 루프(while true) 코드가 여러 개가 따로 돌면 서로 충돌하거나 프로그램이 꼬일 수 있다는 것을 배웠다.

### 느낀 점:
생각한 대로 블록이 멈추지 않거나 무한 루프 때문에 먹통이 될 때 답답하기도 했지만, 코드를 수정해서 블록이 움직이게 만들었을 때 성취감을 느꼈다. 

### 막힌 점:
1. 블록이 떨어지지 않았다. -while true 코드가 2개가 있었는데 한 코드 안에 넣으니 잘 실행되었다.
2. 블록이 떨어졌을 때 블록이 멈추게 말들려 했는데 실행이 되지 않았다. - 블록이 가속도를 받으며 떨어지게 만들었던 것을 키를 누르면 위치가 변하도록 만들었다.
3. 블록이 떨어졌을 때 계속 움직였다.
